# вҡҷпёҸ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/DreamNest-AI.git
cd DreamNest-AI
```

### Install frontend

```bash
cd frontend
npm install
npm run dev
```

### Install backend

```bash
cd backend
npm install
npm run dev
```
